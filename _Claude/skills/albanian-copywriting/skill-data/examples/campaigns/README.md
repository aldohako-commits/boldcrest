# Campaign Examples

> Add past campaign copy here as individual files.
> Name files clearly: `[client]-[campaign-name]-[year].md`
>
> Example: `automita-drive-smarter-2024.md`
>
> Each file should contain:
> - The brief (1-2 sentences)
> - The copy produced (headlines, body, CTAs)
> - Any notes on what worked or what the client changed
>
> These files are used as few-shot models when writing new copy for the same client or category.
